창윤이뱃살

창윤이 뱃살은 심각한 내장비만과 피하지방이 병행되어 있다.

html문서의 특징과 주의사항

  • 시멘틱언어, 웹표준, 웹접근성, 크로스브라우징을 준수해야함
  • dom의구조(문서구조): 엘리먼트의 집합
  • 엘리먼트 : 태그네임, attribute(속성)
  • dom은 정해진 노드(부모,자식,형제,조상,자손)를 갖는다
가장중요한내용만넣기

다양한 시멘틱 요소

  • h1-6(heding) : h1요소는 한개만 포함하길 권장
  • p(파라그래프) : p요소는 두개이상의 단어가 모인 하나의 문장
  • ul,li : 순서가 없는 목록
  • ol,li : 순서가 존재하는 목록
  • main, section, article : 구획요소
  • em, b, strong : 강조, 중요
  • small : 보조적인 내용
  • i : 구분
  • address : 주소, 이름
  • time : 시간
  • blockquote, q : 인용문
  • cite : 출처
  • code : 프로그램 언어
  • kbd : 키보드
  • sub : 아래첨자
  • sup : 윗첨자
  • div,apan : 의미가 없음
