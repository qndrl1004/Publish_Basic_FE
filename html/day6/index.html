form요소

웹브라우저를 이용해 사용자가 제어할 수 있는 ui

요소의 개념

  • 전송할 데이터의 범위
  • action속성 : 데이터를 전송할 주소
  • method속성
    • get : 주소창을 이용해 텍스트로 데이터를 전송
    • post : 보안,바이너리 파일을 전송

form의 구획요소

  • fieldset : 관련된 데이터의 범위를 설정
  • legend : 필드셋 제목 설정

form의 접근성

  • laber요소 : for과id 속성으로 연결
  • title속성 : label텍스트를 대체할 수 있는 속성

input요소 type

  • text(default) :
  • password
  • email
  • tel
  • ....
  • size(글자크기), maxlength(글자수제한), min(최소수), max(최대수), step(증가량)
    placeholder(입력양식에 대한 레퍼런스 제공), name(), cols(), rows()
    accept(), required(유효성 검사)

입력양식의 종류

  • input
  • select
  • textarea
  • button
