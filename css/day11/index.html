grid layout

  • grid속성은 blocklevel과 유사하다
  • grid의 예시 : repeat(3,(auto,1fr,?px))
  • grid요소의 자식은 크기를 정하지 않을시 최소 그리드 하나 가진다
  • 자식요소의 grid칸 설정 : grid-column 2/4(2이상4미만),2/ span 2
  • grid-column: 3칸에서 시작되는 1칸짜리
  • grid-area: 줄시작/칸시작/줄끝/칸끝
  • 노드순서에 영향을 받지않고 레이아웃을 구현할 수 있다
