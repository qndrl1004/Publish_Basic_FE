display level

display의 종류

block level

  • 한줄을 통째로 차지하는 요소
  • 기본적으로 부모의 넓이와 동일한 넓이를 가지게됨
  • 크기 속성을 가질수 있다

inline level

  • 줄내림이 되지 않는 요소
  • 크기속성을 가질 수 없다
  • a, small, em, i

inline-block 레벨

  • 줄내림이 되지 않으면서, 크기를 가질 수 있음
  • (inline-block tag) : svg, img, video, iframe, button, input, textarea

inline 레벨간의 수평 줄맞춤

  • inline은 auto값이 불가능 하다.
