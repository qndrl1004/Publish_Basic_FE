선택자

casecading style sheet

특징과 개념

  • html문서를 기반으로 한 디자인 언어
  • 우선순위가 존재한다
  • 상속개념(inheritance)이 존재한다
  • 다양한 선택자(selecter)가 존재한다

선택자(selector)

태그선택자(type selecter)

  • style엘리먼트 안쪽에서 tagname{속성(property):속성값(value)}
  • 모든 태그에 동일한 스타일이 적용된다.

class selecter

  • style요소안에서 .클래스네임{속성:속성값} 선언후 html요소의 속성 class 속성으로 호출
  • 우선순위 : 클래스, 태그선택자
  • 반복해서 호출이 가능하다.(수정 관리가 용이하다.)

아이디 선택자

  • style요소내에서 #아이디{속성:속성값}선언후, html id속성으로 호출
  • id는 중복호출 불가하다.
  • 우선순위 : 아이디,클래스

inline style

  • html요소의 style속성을 추가후 css문법을 작성
  • 선택자들보다 우선순위가 높다.
  • script로 스타일을 제어할때 사용된다.
  • !important

우선순위

  • 속성: 속성값 !important;
  • inline style
  • id selector
  • class selector
  • type selector
  • chidren selector
