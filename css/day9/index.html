좌표를 이용해서 특정위치로 요소 배치

absolute

  • abs 포지션이 적용되면 독립적인 위치를 가진다
    다른요소와 겹치고, 부모는 높이를 인식할수 없다
  • 포지션이 적용되면 좌표를 설정할 수 있다 : left,right,top,bottom(부모기준으로 움직인다,+와- 잘사용하기)
  • absolute의 좌표 기준점은 position 가지고있는 부모기준이다

fixed와 absolute의 차이

  • absolute는 문서기준 fixed는 브라우저와 붙어있다
  • fixed의 좌표는 부모와 상관없다
    (단 부모요소에 transform속성이 적용되면 예외)

relative

  • absolute가 적용된 자식요소의 좌표 기준점이 된다.
  • 현재 레이아웃을 그대로 유지하면서 position속성을 적용할 수 있음
