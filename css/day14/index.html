transition

  • 숫자값으로 설정된 css속성의 전환효과
  • transition:속성1 초1s,속성2 초s,
    transition:all 초s
  • transition:속성1 초s,속성2 초s transition-delay:0s 1s
  • transition-timing-function 감가속; ease-out(감속) ease-in(가속) linear(동속)

animation

  • 복잡한 모션 효과 구현
  • @keyframes을 이용해서 애니메이션 설계
    animation속성을 이용해서 에니메이션 호출
  • animation-name: move;키프레임 이름
    animation-duration: 3s;애니메이션 시간
    animation-iteration-count: infinite;반복횟수
    animation-direction: alternate;감가속
    animation-timing-function: linear;키프레임진행방향
  • animation이 중단시 자연스럽지 못하다(전환효과가부자연스럽다)
