다양한 선택자

전체선택자(univaersial selector)

*{} : 모든 엘리먼트를 의미한다.

combine selector

div.box : box라는 이명을 가진 div
div#box.box : 아이가 box이면서 클래스는 box인 div
:hover : *가 생략되고 모든게 호버효과

Adjacent Sibling Selector

div + : 태그 뒤에있는 하나만 된다.
div ~ * : 태그 뒤에있는 모든것
div ~ p : 태그 뒤에 선택된 태그들

가상 셀렉터

:first-child : 모든요소 순서를 기준으로 첫번째인 div찾는다. div:first-child 0개
div:first-of-type 맨 앞에 div
p,div,div,div,a,span

:first-of-type : div순서를 기준으로 첫번째 태그

:last-child : 마지막에 있는 태그를 선택

:last-of-type : div순서를 기준으로 마지막 태그

가상 클래스

:nth-child(n) : n번째 태그
:nth-of-type(n) : 태그중 n번째
p,p,div,div,a,a,

div:nth-child(even) : 모든요소 기준으로 짝수 번째
div:nth-of-type(even) : div 기준으로 짝수 번째 div
a,div,div,p,div,div
div:nth-child(odd) : 모든요소 기준으로 홀수 번째
div:nth-of-type(odd) : div 기준으로 홀수 번째 div
